
# SP9820E_Magisk

在sp9820e设备上不修改boot即可使用Magisk
- 前排提醒:你需要有root权限和adb连接才能使用此项目！！！
- Maigsk版本:22.1
## 使用教程

打开install-magisk.bat按照提示安装
